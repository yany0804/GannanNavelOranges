# Vue 3 + TypeScript + Vite

# 🍊 “赣橙橙” —— 专售赣南脐橙商城

> 基于 Vue 3 + Vite + Node.js (Express) + MySQL 的前后端分离电商平台。

## 📖 项目介绍

本项目是一个专注于赣南脐橙销售的垂直电商平台——**“赣橙橙”**。项目旨在利用现代 Web 开发技术，为用户提供一个便捷、美观的在线购买渠道，推广国家地理标志保护产品——赣南脐橙。

系统采用前后端分离架构设计：
* **前端**：构建了响应式的用户界面，包含首页展示、商品筛选、购物车管理等模块，提供流畅的交互体验。
* **后端**：提供 RESTful API 接口，处理用户认证、商品数据查询及订单逻辑。

## 🛠️ 技术栈 (Tech Stack)

### 前端 (Frontend)
* **核心框架**: Vue 3 (Composition API)
* **构建工具**: Vite
* **路由管理**: Vue Router 4 (History 模式)
* **状态管理**: Vuex 4 (管理购物车及用户登录状态)
* **UI 组件库**: Element Plus
* **HTTP 请求**: Fetch API / Axios

### 后端 (Backend)
* **运行环境**: Node.js
* **Web 框架**: Express
* **数据库**: MySQL 8.0+
* **ORM/驱动**: mysql2 (连接池管理)

## 🧩 主要功能模块

### 1. 🏠 首页 (Home)
* **轮播图展示**: 展示赣南脐橙产地风光及最新活动。
* **特性服务**: 展示“全国包邮”、“品质保证”、“新鲜速达”、“售后无忧”四大服务承诺。
* **精选商品**: 推荐热门脐橙礼盒，支持快速加入购物车。

### 2. 🛍️ 商品浏览 (Products)
* **多维筛选**: 支持按**价格区间**（0-50元, 50-100元等）筛选商品。
* **排序功能**: 支持按价格从低到高或从高到低排序。
* **精品专区**: 展示高端礼盒及有机认证产品，支持弹窗查看详细参数（产地、规格、口感）。

### 3. 🛒 购物车 (Cart)
* **抽屉式交互**: 采用右侧抽屉式设计，不打断浏览体验。
* **即时计算**: 实时计算选中商品的总金额。
* **数量管理**: 支持对商品数量进行增减操作。

### 4. 👤 用户中心 (User)
* **注册/登录**: 提供完整的用户注册流程（包含表单验证）及登录功能。
* **导航栏高亮**: 根据当前路由自动高亮显示对应的导航菜单。

### 5. ℹ️ 关于我们 (About)
* 介绍赣南脐橙的历史文化、种植特点及荣誉认证（如“国家地理标志保护产品”）。

## 🚀 快速启动 (Quick Start)

### 1. 数据库准备
1.  确保本地已安装 MySQL (8.0+)。
2.  执行 `db/init.sql` 脚本，创建数据库 `gannan_mall` 及相关表结构。
3.  *(可选)* 修改 `config/database.js` 或 `.env` 中的数据库连接配置。

### 2. 启动后端服务
```bash
# 进入后端目录
cd server

# 安装依赖
npm install

# 启动服务
node index.js
```

### 3. 启动前端服务
```Bash

# 安装依赖
npm install

# 启动开发服务器
npm run dev
打开浏览器访问控制台输出的地址即可预览项目。
```
